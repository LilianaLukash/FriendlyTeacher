# Сборка Android (APK) для установки без компьютера

Используется **EAS Build** (облако Expo). Нужен аккаунт на [expo.dev](https://expo.dev) (бесплатно).

## 1. Один раз: установить CLI и войти

```bash
npm install -g eas-cli
eas login
```

## 2. Привязать проект к Expo (один раз)

В корне проекта:

```bash
cd путь/к/FriendlyTeacher
eas init
```

Если спросит — создай новый проект или привяжи к существующему.

## 3. Адрес API для сборки

В облаке **нет** твоего `.env`. Задай URL Railway так:

**Вариант A — секрет (удобно, URL не в Git):**

```bash
eas secret:create --scope project --name EXPO_PUBLIC_API_URL --value https://friendlyteacher-production.up.railway.app --type string
```

URL без `/` в конце.

**Вариант B — в `eas.json` уже задан `env` для профилей сборки** (можно переопределить секретом выше).

## 4. Собрать APK

```bash
eas build --platform android --profile production
```

Следуй вопросам в терминале (ключ Android: можно сгенерировать новый через EAS).

## 5. Скачать и передать маме

- Когда сборка закончится, в браузере откроется страница билда или ссылка придёт в консоль.
- Скачай **.apk** и отправь маме (Telegram, Google Drive, почта).
- На телефоне: **Настройки → Безопасность → разрешить установку из неизвестных источников** (или по запросу при открытии APK) → открыть файл и установить.

## 6. Обновления

После изменений в коде снова:

```bash
eas build --platform android --profile production
```

И снова отправь новый APK (или настрой позже обновление через Play Store — отдельная история).

---

**Важно:** на Railway API должен быть онлайн и с теми же переменными (`OPENAI_API_KEY` и т.д.), что и раньше.
